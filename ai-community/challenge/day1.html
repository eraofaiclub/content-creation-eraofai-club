01

Step-by-Step Setup

1
Install OpenClaw
Clone the repo, install dependencies, and run it locally. Works on Mac, Windows, and Linux. Takes under 5 minutes.
2
Create a Discord Bot
Head to the Discord Developer Portal, create a new application, add a bot, and grab your token. We'll walk through every click.
3
Configure OpenClaw for Discord
Add your Discord bot token and channel IDs to OpenClaw's config. Set up which channels the agent listens to and how it responds.
4
Set the Agent's Personality
Write a system prompt that defines how your agent behaves — helpful, concise, on-brand. This is where the magic happens.
5
Go Live
Invite the bot to your server, send your first message, and watch your AI agent respond. You just built something real on Day 1.
